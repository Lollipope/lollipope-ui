    # 代码

    # Attributes

    参数 说明 类型 是否必要 可选值 默认值
    type 类型 String false primary / success / warning / danger / info / text -
    plain 是否朴素按钮 boolean true - -
    round 是否圆角按钮 boolean true - -
    circle 是否圆形按钮 boolean true - -
    disabled 是否禁用状态 boolean true - -
